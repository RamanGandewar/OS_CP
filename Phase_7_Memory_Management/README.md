# Phase 7 - Memory Management

This phase adds memory-management concepts to the Sales CRM:

- paging for large report access
- page replacement comparison using FIFO, LRU, and Optimal
- LRU caches for customer, quotation, and user preference data
- memory allocation strategies using First Fit, Best Fit, Worst Fit, and Next Fit
- a Memory Monitor dashboard in the frontend

Run the full CRM from the project root:

```bash
python CRM.py
```
