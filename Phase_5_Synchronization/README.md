# Phase 5 - Synchronization

This phase adds synchronization primitives and concurrency conflict monitoring to the CRM.

## Included

- Mutex for concurrent quotation editing
- Semaphore for stock/report resource limits
- Reader-writer lock for report access
- Producer-consumer bounded buffer
- Race condition before/after demonstration
- Synchronization monitor dashboard

Run the CRM from the project root with:

```bash
python CRM.py
```
