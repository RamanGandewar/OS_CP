# API Endpoint Documentation

## Authentication

- `POST /api/auth/login`
- `POST /api/auth/logout`
- `GET /api/auth/me`

## Master Data

- `GET /api/users`
- `GET /api/customers`
- `POST /api/customers`

- `DELETE /api/customers/<id>`
