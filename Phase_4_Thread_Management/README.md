# Phase 4 - Thread Management

This phase extends the CRM with multi-threading concepts:

- daemon threads
- timer threads
- worker threads
- thread pool execution
- thread-safe monitoring

## Main Files

- `code/thread_manager.py`
- `code/thread_pool.py`
- `code/thread_monitor.py`
- `code/routes/thread_routes.py`
- `code/thread_validation_test.py`

Run the CRM from the project root with:

```bash
python CRM.py
```
