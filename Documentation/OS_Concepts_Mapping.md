# OS Concepts Mapping

## Mapping Summary

- Process management: user sessions and background child processes
- CPU scheduling: CRM task queue and algorithm comparison
- Thread management: daemon threads, worker pool, validation/report jobs
- Synchronization: locks, semaphores, reader-writer coordination
- Deadlock handling: resource allocation graph, Banker checks, recovery
- Memory management: paging, replacement, caching, allocation strategies
- I/O management: disk scheduling, buffering, print spooling
