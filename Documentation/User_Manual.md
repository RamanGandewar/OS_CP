# User Manual

## Installation

1. Create and activate the backend virtual environment.
2. Install backend dependencies with `pip install -r Backend\requirements.txt`.
3. Install frontend dependencies with `npm install` inside `Frontend`.
4. Seed the database with `python Backend\database\seed_data.py`.

## Running the Application

Run from the project root:

```bash
python CRM.py
```

## Using the Unified Dashboard

- Open the main dashboard tab to see overall system health.
- Switch between Processes, Scheduler, Threads, Synchronization, Deadlock, Memory, and I/O tabs.
- Use export buttons for CSV and report output.
