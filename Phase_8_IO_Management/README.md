# Phase 8 - I/O Management

This phase adds disk scheduling and file-I/O concepts to the Sales CRM:

- disk scheduling with FCFS, SSTF, SCAN, C-SCAN, LOOK, and C-LOOK
- seek time comparison and disk head movement visualization
- I/O buffer pool management
- print-job spooling for invoices and reports
- a Disk Scheduler Monitor page in the frontend

Run the full CRM from the project root:

```bash
python CRM.py
```
