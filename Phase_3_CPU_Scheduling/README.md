# Phase 3 - CPU Scheduling

This phase extends the CRM with classic CPU scheduling algorithms for background task management.

## Algorithms Included

- FCFS
- SJF
- Priority Scheduling
- Round Robin

## Main Files

- `code/task.py`
- `code/scheduler.py`
- `code/gantt_generator.py`
- `code/performance_analyzer.py`
- `code/routes/scheduler_routes.py`
- `code/scheduler_validation_test.py`

## Run

Start the full application from the project root:

```bash
python run_phase_2.py
```

Then open the `CPU Scheduler` page in the frontend.
